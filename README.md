# Knight's Ascent

## Objectif du jeu
Le jeu est un plateformer 2D dans lequel vous incarner un chevalier qui a pour objectif de tuer un sorcier

## Capacités du joueur
 - Il possede une attaque avec son épée
 - Il peut effectuer un dash (déplacement rapide dans une direction sans être impacté par la gravité) qui le rend invulnerable aux ennemis
 - Il peut effectuer une roulade (déplacement dans une direction plus lent que le dash) qui le rend aussi invulnerable

## Le sorcier
Le sorcier est le boss final du jeu, il possède lui aussi un dash, de plus il possede plusieurs type d'attaque qu'il débloque tout le long du combat :
 - Phase 1 : boule de feu
 - Phase 2 (vie<60%) : Vague d'éclairs qui vont dans une direction
 - Phase 3 (vie<30%) : Une météorite

## Les grenouilles
les grenouilles sont les enemis par défaut qui tuent au touché, ils ont des loot quand ils sont tué :
-  5% : avoir une vie en plus
- 10% : avoir une pièce d'or qui vaut 10
- 35% : avoir une pièce d'argent qui vaut 5
- 45% : avoir une pièce de bronze qui vaut 1

## Test du jeu
Pour pouvoir voir les 3 phases du boss, on peut devenir invincible avec la touche k
